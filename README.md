♻️ ecoCycle: 분리수거 안내 서비스

  Java Swing으로 시작하여 Spring Boot 기반의 웹 서비스까지 확장을 하였습니다.

📺 프로젝트 개요

  이 프로젝트는 기존의 정적인 분리배출 안내 방식에서 벗어나, 퀴즈와 랭킹 시스템을 결합하여 사용자의 자발적인 참여를 유도를 합니다.

✨ 주요 기능
1. 분리수거 퀴즈 
지식 습득: 재활용 상식을 퀴즈 형식으로 풀며 올바른 배출 방법을 학습합니다.

  즉각적 보상: 정답 시 포인트를 지급하여 참여 동기를 부여합니다. (1일 1회 제한 로직 적용)

2. 랭킹 시스템 
경쟁 요소: MySQL DB 연동을 통해 실시간 사용자를 시각화하여 경쟁 심리를 자극합니다.

  성취감: 환경 기여도를 포인트화하여 자신의 순위를 확인할 수 있습니다.

3. 분리수거 가이드 
이미지 로딩: 품목별 상세 요령을 시각적 이미지와 함께 제공합니다.

  통합 검색: 궁금한 품목의 배출 방법을 즉시 검색할 수 있습니다.

4. 가상 스토어 및 등급
성장 시스템: 포인트에 따라 등급이 변화하는 시스템.

  자원 순환: 획득한 포인트로 보상을 얻는 가상 경험을 제공합니다.
